# Rei Gelato - Sacolé Gourmet Artesanal

## 🌟 Sobre o Projeto
O **Rei Gelato** é um site moderno e responsivo criado para uma loja de sacolés gourmet artesanais. O objetivo é proporcionar uma experiência visualmente agradável e funcional para os clientes, permitindo que eles naveguem facilmente pelo cardápio, conheçam a história da marca e entrem em contato via WhatsApp.

### ✨ Principais Funcionalidades
- **Menu Responsivo:** Um menu hambúrguer adaptado para dispositivos móveis e tablets, garantindo uma navegação fácil.
- **Scroll Suave:** Transições suaves ao clicar nos links do menu, melhorando a experiência do usuário.
- **Integração com WhatsApp:** Botões "Pedir Agora" e formulário de contato que abrem conversas no WhatsApp com mensagens pré-definidas.
- **Design Moderno:** Interface limpa e profissional, com cores e fontes que refletem a identidade visual da marca.
- **Catálogo de Produtos:** Exibição organizada dos sabores disponíveis, com imagens e preços.

### 📱 Responsividade
O site foi desenvolvido para funcionar perfeitamente em todos os dispositivos:
- **Celulares:** Menu hambúrguer e layout otimizado para telas pequenas.
- **Tablets:** Design adaptável para telas médias.
- **Desktops:** Layout completo com menu horizontal tradicional.

### 🛠️ Tecnologias Utilizadas
- **HTML5:** Estrutura do site.
- **CSS3:** Estilização e design responsivo.
- **JavaScript:** Interações dinâmicas, como o menu hambúrguer e scroll suave.
- **Font Awesome:** Ícones modernos e personalizados.
- **Google Fonts:** Tipografia elegante e acessível.

### 🚀 Como Acessar o Site
O site está hospedado na Netlify e pode ser acessado através do seguinte link: https://reigelato.netlify.app
